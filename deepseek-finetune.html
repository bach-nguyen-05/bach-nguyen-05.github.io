DeepSeek-R1 Fine-Tune and Benchmark

This project involved fine-tuning the DeepSeek-R1 model on a reasoning data. Low Rank Adaptation (LoRA) is used to efficiently train the model and benchmarking using "biobertscore" model on the rest of the dataset that was not used to train.

Technologies Used: Python, PyTorch, Hugging Face Transformers.

Key Achievements:

  • Achieved a 0.5% improvement in accuracy on the benchmark dataset.
  • Optimized the model for faster inference times.
  • Saved post-trained model in gguf and run on an Android phone locally
